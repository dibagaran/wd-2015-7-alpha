• 'strong' tag defines important text.
  • 'em' tag defines emphasized text.
  • 'abbr' defines an abbreviation or an acronym like LHC.
  • 'address' defines contact information like:
    John Doe - Box 564, Disneyland, USA
  • 'blockquote' defines a section that is quoted from another source like:
    The first principle is that you must not fool yourself and you are the easiest person to fool.
  • 'q' defines a short quotation usually in the context of the current text like:

    And AC said, LET THERE BE LIGHT!

  • 'cite' defines citation or the title of a work like: A Universe from Nothing by Lawrence M. Krauss
  • 'code 'defines a piece of computer code like: body { padding: 0; margin: 0; }
  • 'kbd' defines a keyboard shortcut or input like: Ctrl+N
  • 'ul' defines an unordered list like:
    • Item 1
    • Item 2
    • Item 3
  • 'ol' defines an ordered list like:
    1. Item 1
    2. Item 2
    3. Item 3
  • 'pre' defines preformatted text like: 
    Text in a pre element
preserves
both      spaces and
line breaks
  • 'span' defines a generic container which does not inherently represent anything. It can be used to group elements for styling purposes.
  • 'sub' defines subscripted text like X0
  • 'sup' defines superscripted text like X2
